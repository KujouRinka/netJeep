### netJeep

A simple extensible asynchronous network framework.

#### requirements

[Boost](https://www.boost.org/) >= 1.66.0

[Asio non-Boost](https://think-async.com/Asio/) == 1.22.1

[CryptoPP](https://www.cryptopp.com/) == 8.6
